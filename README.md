# Курсовой проект по предмету "Проектный семинар с фреймворками JavaScript"

Результат доступен в онлайне на [vercel.app](https://vuejs-eshop.vercel.app/).

## Локальный запуск проекта

Установить зависимости проекта (сборка тестировалась с NodeJS версии 22.3):

```bash
pnpm i
```

Запустить локальный бэкенд, при необходимости (отличный от умолчания URL) прописать адрес бэкенда в параметре `VITE_API_BASE_URL` в .env.local, далее:

```bash
pnpm build # запуск в prod-режиме 
pnpm dev # запуск в dev-режиме
```
